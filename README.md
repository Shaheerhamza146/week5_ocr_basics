# OCR Basics - Document Intelligence

This project demonstrates Optical Character Recognition (OCR) using:

- Tesseract OCR
- EasyOCR
- OpenCV preprocessing

## Features
- Text extraction from receipts
- Image preprocessing (grayscale, blur, thresholding)
- OCR comparison
- Receipt data parsing (total extraction)

## Results
OCR worked successfully with preprocessing improvements. Some limitations were observed due to synthetic image quality.

## Tools Used
- Python
- OpenCV
- Tesseract
- EasyOCR
